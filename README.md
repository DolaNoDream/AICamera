# "瞬景Pose" AiCamera

## 简介

一款集成AI 智能算法与多端协同能力的智能相机项目，核心围绕安卓端相机能力打造，融合姿势推荐、语音交互（STT/TTS）、前后端数据互通等功能，形成「安卓客户端 - 多版本后端 - 标准化 API」的全链路 AI 相机解决方案。

## 核心功能

-📷 安卓端相机流采集与管理，适配原生相机开发规范
-🤖 AI 姿势推荐：拍照 / 录像场景下的智能姿势指导（配套专属 API）
-🗣️ 语音交互：集成 STT（语音转文字）、TTS（文字转语音）能力
-🔗 标准化前后端接口：提供完整的接口文档，支持多端数据互通
-🧩 多版本后端服务：适配不同场景的后端实现，灵活扩展
